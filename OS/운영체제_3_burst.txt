
CPU (central processing unit)
    프로세스의 명령어를 해석하고 실행하는 장치

IO (Input/output)    
    파일을 읽고 쓰거나 네트워크의 어딘가와 데이터를 주고 받는것
    입출력 장치와 데이터를 주거나 받는 것

버스트 (Burst)
    어떤 현상이 짧은 시간 안에 집중적으로 일어나는 일

CPU 버스트  
    프로세스가 CPU에서 한번에 연속적으로 실행되는 시간

IO 버스트
    프로세스가 IO작업을 요청하고 결과를 기다리는 시간

** 프로세스는 CPU 버스트와 IO 버스트의 연속

CPU BOUND 프로세스란
    CPU 버스트가 많은 프로세스
    EX) 동영상 편집 프로그램, 머신러닝 프로그램

    적절한 스레드 수는 CPU수 + 1
        - 스레드가 너무 많아 컨텍스트스위칭이 자주 발행하면 오히려 불필요하게 CPU자원을 낭비하는셈
        
IO BOUND 프로세스란
    IO 버스트가 많은 프로세스
    EX) (일반적인) 백엔드 API 서버
    
    적절한 스레드 수는 서비스에 따라 다름
        - 컴퓨터 스팩, 프로그램의 특성, 서비스의 방식, 예상되는 그래픽의 패턴은 어떤지 등등등
